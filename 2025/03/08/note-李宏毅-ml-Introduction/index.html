预测第二天点阅次数

  1. Function with Unknown Parameters

y = b+wx

  1. Define Loss from Training Data
  • Loss is a function of parameters L(b,w)
  • Loss: how good a set of values is.
  1. Optimization
  • 使得 Loss 最小的 参数组合
  • Gradient Descent

使用大量的 hard sigmoid 相加逼近某一个 函数
$$
count(update)\times count(batch_size) = count(example_num) = 1 epoch
$$

训练攻略

训练集上的 loss 大

  • Model Bias
    • 模型太简单了
    • 不管 $\theta$ 取多少都没办法 减小 $loss$
    • 增加模型的 feature 数目，使 模型更复杂
  • Optimization
    • 存在一些 $\theta$ 的取值，使得 loss 变小，但是当前的优化方法无法找到这样的 $\theta$
    • local minima

区分 Mode Bias 和 Optimization Issue

  • 20-layer 对应Loss在测试集上更低，表明不是Overfitting
  • 在 Training Data 上，20-layer更低，表明不是 Model-Bias 因为 层数更多，Loss反而变高，不是模型复杂度不够

训练集上的 loss 小，但 测试集上的 loss 大

  • overfitting
    • 增加训练资料
      • Data Augmentation
        • 根据对问题的认知，人为的创造出新的训练资料
          • 影像辨识，将图片左右翻转 截取一部分放大
    • 给模型加上一些限制
      • less parameters(减少神经元数目)
      • sharing parameters
      • less feature
      • early stopping
      • regularization
      • dropout

模型过于复杂时，产生 overfitting

避免在 public Testing Set 上调整模型，可能导致模型 overfitting 在 public Testing Set 上

划分训练集 and 测试集

  1. 随机分
  2. N-fold Cross Validation
    • 轮流作为 Train Set

mismatch

Testing Set 上的资料出现了异常，分布和 Training Set 不同
