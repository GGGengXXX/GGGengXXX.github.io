What is Transformer?

  • A sequence to sequence model

What can a seq2seq model do?

common: Language Translation

  • input: voice output: text
  • input: language1 output: language2
  • add above two together
    • input: voice output: text in another language
    • why? some language do not have text, we need to transform it into another language
  • job done by seq2seq
    • QA(Question Answer)
    • Syntactic parsing (文法划分)
      • 输出树结构 – 用广义表 （+括号）
    • Multi-label Classification(多标签分类)
    • Object Detection

How to manage it

basic structure

Transformer Encoder

  • 每个block都是一套相当复杂的过程
  • Bert 使用了 Transformer 的 Encoder 模型

Related Essays

  • On Layer Normalization in the Transformer Architecture

    • https://arxiv.org/abs/2002.04745
    • A new Structure for block was used

  • PowerNattorm: Rethinking Batch Normalization in Transformers

    • https://arxiv.org/abs/2003/07845

Transformer Decoder

  • Autoregressive

Masked Self-attention

  • 只考虑前面的，不考虑后面的输入
    • 在 产生 b1 的时候 a2 a3 a4 还没有产生

如何终止输入

  • 设置一个特殊的中断符号

AT vs NAT(Non-Autoregressive)

NAT实现并行处理，同时吃多个 BEGIN 作为input

NAT终止输入

  • 设置一个额外的Classification 用于决定input的数目

  • 给定input数目，将output 中 中断符号后的全部舍弃

  • Generally AT performs better

cross attention

Training

  • 输出 和 目标 One-hot vector 越接近越好
  • Teacher Forcing : using the ground truth as input
    • 向 encoder 中输入正确答案

Copy Mechanism

  • Chat-bot
    • User: 你好 我是 库洛洛
    • Machine: 库洛洛 你好 很高兴认识你、
  • 从输入中复制一些东西出来 作为输出

Beam Search

  • Not possible to check all the path
  • 需要创造力的任务 无法perform well
  • 需要加一些noise， （语音合成 TTS

Related Research

  • When you do not know how to optimize, just use reinforcement learning(https://arxiv.org/abs/1511.06732)

Exposure Bias

  • 在训练的时候 Decoder 接受的是正确的答案作为input

  • 在测试的时候 Decoder 看到自己的输出， 存在错误

  • solution （Scheduled Sampling）

    • 给Decoder 训练时 加入错误的输入
  • Original Scheduled Sampling
    • 1506.03099
  • Scheduled Sampling for Transformer
    • 1906.04331
  • Parallel Scheduled Sampling
    • 1906.04331
